# 🎮 Gaming Event Management Website

A modern **gaming event management platform** designed to organize and manage competitive tournaments for popular games like **Free Fire, PUBG Mobile, Call of Duty, and Valorant**.
The system supports **game-based registrations, team management, event listings, and responsive UI** for a smooth player experience.

---

## ✨ Key Features

### 🏠 Home Page

* Eye-catching **hero section** with call-to-action buttons
* **Featured games** showcase (Free Fire, PUBG, COD, Valorant)
* **Upcoming tournaments** with dates, prize pools, and registration access
* Platform highlights:

  * Live updates
  * Fair play system
  * Leaderboards
  * Exciting prizes
* **Statistics dashboard**:

  * Active players
  * Total tournaments
  * Prize pools
  * Registered teams
* Fully **responsive design** (desktop, tablet, mobile)

---

## 📝 Game-Based Registration System

* 🎯 **Game selection** before registration
* 🧩 **Dynamic forms** based on the selected game
* 👥 **Team & Solo registration support**
* 🔍 Built-in **form validation** with visual feedback
* 🔄 Smooth navigation between pages

### Supported Game Modes

| Game                | Modes                |
| ------------------- | -------------------- |
| **Free Fire** 🔥    | Solo, Duo, Squad (4) |
| **PUBG Mobile** 🎯  | Solo, Duo, Squad (4) |
| **Call of Duty** ⚔️ | Solo, 2v2, 5v5       |
| **Valorant** 💎     | 5v5 Team             |

### Player & Team Details

* Player name
* Email address
* Phone number
* In-game ID
* Team name
* Team size / mode
* Experience level
